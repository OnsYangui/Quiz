Quiz : HTML, CSS et JavaScript

1. Que signifie HTML ?

a) Hyper Text Markup Language

2. Quelle balise utilise-t-on pour insérer un paragraphe ?

a) <par>

c) <pg>

3. Quel attribut CSS permet de changer la couleur du texte ?

a) text-style
b) font-color
c) color

4. Quelle est la signification de CSS ?

a) Creative Style Sheets
b) Computer Style Sheets
c) Cascading Style Sheets

5. Quel est le rôle de JavaScript ?

a) Créer du contenu statique

c) Structurer les données

a) <img>

c) <pic>

7. Comment déclare-t-on une variable en JavaScript ?

a) var
b) variable
c) v

8. Quel est l'opérateur d'affectation en JavaScript ?

a) =
b) ==
c) ===

9. Quelle pseudo-classe CSS est utilisée pour styliser un lien survolé par la souris ?

a) :visited
b) :active
c) :hover

10. Quel sélecteur CSS est utilisé pour sélectionner tous les éléments avec une classe spécifique ?

a) #classname
b) .classname
c) *classname

11. Quelle est la valeur par défaut de la propriété position en CSS ?

a) static
b) relative
c) fixed

12. Que signifie l'attribut alt dans une balise <img> ?

a) Ajouter une bordure à l'image
b) Fournir un texte alternatif si l'image ne peut
c) Changer la taille de l'image
